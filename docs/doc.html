Getting Started

Follow the following steps to get started with yorlang

Steps:

npm install -g yorlang

Create test.yl file.

Write 'sọpé "báwo ni ayé" '>

Run yorl test.yl in your cli

Variable Declaration

Variables can be declared in yorlang using

Yorlang: tí

Variables in yorlang must be initialized and declared immediately. Variables in yorlang can also be reassigned another value even if it had been declared before with the condition that the keyword (tí) is also used. strings in yorlang must be written using double quotes. yorlang variables can hold many data types such as strings, integers and booleans

Note:
comments can be made in yorlang using #. There's no multiline comments like /** */
A semi colon must be used at the end of every statement in yorlang

yorlang

tí name = "dammy;

tí age = 5 ;

sọpé name + "is" + age + "years old" ;

OUTPUT

dammy is 5 years old

Another example:

yorlang

tí name = "dammy;

tí name = "michelle" ; # variables can be reassigned as long as the keyword (tí ) is used

sọpé "my name is" + name ;

OUTPUT

my name is michelle

Conditionals

Conditionals such as if, else and switch statements can also be used in yorlang

if: ṣé   else: tàbí

Use the if statement (ṣé) to specify a block of yorlang code to be executed if a condition is true

Use the else statement (tàbí) to specify a block of yorlang code to be executed if a condition is false

yorlang

tí time = 15;

ṣé ( time < 12 )  {
    greeting = "good morning" ;

}   tàbí {
    greeting = "good day" ; }

sọpé greeting + "world" ;

OUTPUT

good day world


The Switch Condition:

This is how it works:
  The switch expression is evaluated once.
  The value of the expression is compared with the values of each case.
  If there is a match, the associated block of code is executed and then it escapes the switch
  statement witthout the need for a break (kúrò) expression
  The use of the break expression(kúrò) in yorlang results in an error.

Switch: yí
Case: ẹjó
Default: padàsí
Break: kúrò

yorlang

tí age = 10 + 4 ;

yí ( age )  {
    ẹjó 12 :
        sọpé "you are a kid" ;
        ẹjó 18 :
        sọpé "you are a teenager" ;
        ẹjó 22 :
        sọpé "you are an adult" ;
        padàsí;
        sọpé "you are too old" ;
}

OUTPUT

you are a teenager
