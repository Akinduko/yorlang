Getting Started

Follow the following steps to get started with yorlang

Steps:

npm install -g yorlang

Create test.yl file.

Write 'sọpé "báwo ni ayé" '>

Run yorl test.yl in your cli

Variable Declaration

Variables can be declared in yorlang using

Yorlang: tí

Variables in yorlang must be initialized at the point of declaration. Variables in yorlang can also be reassigned another value even if it had been declared before with the condition that the keyword (tí) is also used. strings in yorlang must be written using double quotes. yorlang variables can hold many data types such as strings, integers and booleans

Note:
comments can be made in yorlang using #. There's no multiline comments like /** */
A semi colon must be used at the end of every statement in yorlang

yorlang

tí name = "dammy";

tí age = 5 ;

sọpé name + " " + "is" + " " + age + " " + "years old" ;

OUTPUT

dammy is 5 years old

Another example:

yorlang

tí oruko = "dammy";

tí oruko = "michelle" ; # variables can be reassigned as long as the keyword (tí ) is used

sọpé "oruko mi ni" + " " + name ;

OUTPUT

oruko mi ni michelle

Conditionals

Conditionals such as if, else, else if and switch statements can also be used in yorlang

if: ṣé     else: tàbí     else if: tàbí ṣé

yorlang

tí time = 15;

ṣé ( time < 12 )  {
    tí greeting = "good morning" ;

} tàbí ṣé ( time > 12 && time < 17 )  {
    tí greeting = "good day" ; }

} tàbí {
    tí greeting = "good evening" ;
}

sọpé greeting + " " + "world" ;

OUTPUT

good day world


The Switch Condition:

This is how it works:
 The switch expression is evaluated once.
  The value of the expression is compared with the values of each case.
  If there is a match, the associated block of code is executed and then it escapes the switch
  block without the need for a break (kúrò) expression.
  The use of the break statement(kúrò) in yorlang yí(switch) condition results in an error.

Switch: yí
Case: ẹjó
Default: padàsí
Break: kúrò

yorlang

tí age = 10 + 8 ;

yí ( age )  {
    ẹjó 12 :
        sọpé "you are a kid" ;
    ẹjó 18 :
        sọpé "you are a teenager" ;
    ẹjó 22 :
        sọpé "you are an adult" ;
    padàsí:
        sọpé "you are too old" ;
}

OUTPUT

you are a teenager

Loops

Loops such as for and while are also supported in Yorlang

For: fún   while: nígbàtí

Note:
increments such as i++ in yorlang is invalid instead use tí i = i + 1;
decrements such as i-- in yorlang is invalid instead use tí i = i - 1;

yorlang

fún ( tí i = 0 ; i < 5 ; tí i = i + 1 ;)   {

sọpé "you can be" + " " + i ;

}

sọpé "you can be anything you want to be" ;

OUTPUT

you can be 1

you can be 2

you can be 3

you can be 4

you can be 5

you can be anything you want to be


The While( nígbàtí) Loop:

The while ( nígbàtí) loop is executed as long as the condition in parentheses is true

yorlang

tí onka = [1, 2, 3, 4, 5 ];

tí i = 0;

nígbàtí ( onka [ i ] < 5) {

    sọpé "o kere si aarun";

    tí i = i + 1;
}

OUTPUT

o kere si aarun

o kere si aarun

o kere si aarun

o kere si aarun

Functions

Functions are also supported in yorlang using the following keywords

Function: iṣé   return: padà

yorlang

iṣé multiplyNumbers ( a, b)   {

    padà a * b ;

}

tí X = multiplyNumbers ( 5, 6 ) ;

sọpé X ;

OUTPUT

30
